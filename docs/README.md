### Self-Service Resources

**Documentation:**

- Use search function (Ctrl+F)

**Video Tutorials:**

- Topics: Login, GPS, Maintenance, Mobile App, Reports, etc.

**Knowledge Base:**
- Access via: Profile Menu → Help & Support → Knowledge Base
- Searchable articles on specific topics

### Contact Support

**Email Support:**
- Email: support@capitaltechalliance.com
- Response time: Within 24 hours (weekdays)
- Best for: Non-urgent issues, detailed questions

**Phone Support:**
- Phone: 1-800-FLEET-HELP
- Hours: Monday-Friday, 8am-5pm EST
- Best for: Urgent issues, real-time troubleshooting

**Emergency Support (System Down):**
- Call main number, press 1 for critical support
- Available: 24/7
- Use only for: System completely down, critical data loss

**When contacting support:**
- Have your tenant name and email ready
- Describe the issue in detail
- Include error messages and screenshots
- Note what you were trying to do
- Mention troubleshooting steps already tried

## Documentation Standards

### For Contributors

**Writing Guidelines:**
- Use clear, simple language
- Avoid technical jargon (or explain it)
- Use active voice ("Click the button" not "The button should be clicked")
- Include screenshots where helpful
- Provide step-by-step instructions
- Test all instructions before publishing

**Formatting:**
- Use Markdown format
- Include table of contents for long documents
- Use descriptive headings
- Use code blocks for commands/code
- Use tables for structured data
- Bold important terms on first use

**Updates:**
- Keep documentation in sync with system changes
- Review and update documentation with each release
- Mark outdated sections clearly
- Version documentation when major changes occur

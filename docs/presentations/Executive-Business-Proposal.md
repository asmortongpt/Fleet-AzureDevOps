## APPENDIX A: GLOSSARY OF TERMS

**AI Agent:** Specialized artificial intelligence algorithm that performs specific optimization or prediction tasks (e.g., fuel efficiency analysis, predictive maintenance).

**API (Application Programming Interface):** Software interface that allows systems to communicate and exchange data.

**DVIR (Driver Vehicle Inspection Report):** Daily inspection form required by DOT for commercial vehicles.

**ERP (Enterprise Resource Planning):** Business process management software (e.g., SAP, Microsoft Dynamics).

**GPS (Global Positioning System):** Satellite-based navigation system for vehicle tracking.

**IFTA (International Fuel Tax Agreement):** Agreement between US states and Canadian provinces for fuel tax reporting.

**KPI (Key Performance Indicator):** Measurable value demonstrating effectiveness of achieving business objectives.

**MPG (Miles Per Gallon):** Fuel efficiency measurement.

**OSHA (Occupational Safety and Health Administration):** Federal agency enforcing workplace safety regulations.

**PM (Preventive Maintenance):** Scheduled maintenance to prevent equipment failure.

**PO (Purchase Order):** Commercial document authorizing a purchase transaction.

**PWA (Progressive Web App):** Web application that functions like a native mobile app.

**RBAC (Role-Based Access Control):** Security approach restricting system access based on user roles.

**RLS (Row-Level Security):** Database security feature limiting data visibility by user/tenant.

**ROI (Return on Investment):** Performance measure calculating benefit relative to cost.

**SaaS (Software as a Service):** Cloud-based software delivery model (subscription-based).

**SLA (Service Level Agreement):** Commitment to service performance levels (e.g., 99.9% uptime).

**TCO (Total Cost of Ownership):** Comprehensive cost analysis including all direct and indirect costs.

**VIN (Vehicle Identification Number):** Unique 17-character vehicle identifier.

## Common Tasks - How To...

### View Vehicle Location
1. Click **"Live GPS Tracking"**
2. Find vehicle on map or search by name/fleet #
3. Click marker for details

### Add a New Vehicle
1. Go to **"Garage & Service"**
2. Click **"+ Add Vehicle"**
3. Enter VIN, license plate, make/model, year
4. Fill in fleet details
5. Click **"Save"**

### Schedule Maintenance
1. Go to **"Maintenance Calendar"**
2. Click **"+ Schedule Service"**
3. Select vehicle and service type
4. Choose date/time and technician
5. Click **"Schedule"**

### Report Maintenance Issue
1. Click **"Maintenance Request"**
2. Select vehicle
3. Describe issue (add photos if needed)
4. Set priority
5. Submit

### Record Fuel Purchase
1. Go to **"Fuel Management"**
2. Click **"+ Add Fuel Entry"**
3. Select vehicle
4. Enter gallons, cost, odometer
5. Upload receipt (optional)
6. Save

### Submit Mileage Reimbursement
1. Go to **"Mileage Reimbursement"**
2. Click **"+ New Trip"**
3. Enter start/end locations and date
4. System calculates IRS mileage rate
5. Add purpose/notes
6. Submit for approval

### Assign Vehicle to Driver
1. Go to vehicle detail page
2. Click **"Assign Driver"**
3. Search and select driver
4. Choose permanent or temporary
5. Confirm

### Generate Report
1. Go to **"Fleet Analytics"** or **"Custom Report Builder"**
2. Select report template
3. Set date range and filters
4. Click **"Generate"**
5. View, export to PDF/Excel, or print

### Send Message to Driver
1. Go to **"Teams Messages"** or **"Communication Log"**
2. Click **"+ New Message"**
3. Select recipient(s)
4. Type message
5. Send

### Optimize Route
1. Go to **"Route Optimization"**
2. Enter stops (addresses)
3. Set parameters (vehicle, time windows)
4. Click **"Optimize"**
5. Send route to driver

## Mobile App Quick Guide

**Download:** Search "Fleet Manager" in App Store or Google Play

### For Drivers

**Start Trip:**
1. Open app → Tap **"Start Trip"**
2. Select vehicle and destination
3. Tap **"Begin"**
4. GPS tracks automatically
5. Tap **"End Trip"** when complete

**Pre-Trip Inspection:**
1. Tap **"Inspection"**
2. Walk through checklist
3. Take photos of any issues
4. Submit

**Report Issue:**
1. Tap **"Report Issue"**
2. Take photo
3. Describe problem
4. Submit (creates maintenance request)

**View Assignments:**
- Tap **"Assignments"** to see scheduled trips/tasks
- Tap any assignment for details and navigation

## Status Indicators

### Vehicle Status Colors
- **Green**: Available
- **Blue**: In Use
- **Yellow**: Maintenance Due
- **Orange**: In Maintenance
- **Red**: Out of Service
- **Gray**: Inactive

### Maintenance Priority
- **Low**: Routine service, can wait
- **Medium**: Should be done soon
- **High**: Needs attention within days
- **Critical**: Immediate action required, vehicle unsafe

### Driver Performance Scores
- **90-100**: Excellent (Green)
- **80-89**: Good (Light Green)
- **70-79**: Fair (Yellow)
- **60-69**: Needs Improvement (Orange)
- **Below 60**: Poor (Red)

## Geofence Alerts

**What they are:** Get notified when vehicles enter/exit specific areas

**Setting up:**
1. Go to **"Geofence Management"**
2. Click **"+ Create Geofence"**
3. Draw boundary on map or enter address + radius
4. Name the geofence
5. Set alert rules (enter/exit/dwell time)
6. Choose who gets alerts
7. Save

## Quick Troubleshooting

| Problem | Quick Fix |
|---------|-----------|
| Can't log in | Reset password or try Microsoft SSO |
| Page loading slow | Refresh page (F5) or clear browser cache |
| GPS not updating | Check GPS device power and cellular signal |
| Email not received | Check spam folder, verify email in profile |
| Data not saving | Check required fields (*), verify internet |

## Monitoring Points

### Key Metrics Tracked

1. **Build Time**
   - Start: ACR build triggered
   - End: Image available in registry
   - Target: < 10 minutes

2. **Deployment Time**
   - Start: kubectl set image
   - End: Rollout complete
   - Target: < 5 minutes

3. **Validation Time**
   - Start: First Playwright test
   - End: All tests complete
   - Target: < 7 minutes

4. **Rollback Time** (if needed)
   - Start: Rollback triggered
   - End: Previous version healthy
   - Target: < 3 minutes

5. **Total Pipeline Time**
   - Success path: 10-22 minutes
   - Failure with rollback: 15-30 minutes

## UI Vision Principles

All components follow these non-negotiable principles:

1. ✅ One primary control surface (not many semi-dashboards)
2. ✅ State before structure (show what's happening first)
3. ✅ One mental model (command center, not tool collection)
4. ✅ Compression over expansion (tabs > pages)
5. ✅ Evidence, not assertions (show WHY, not just status)
6. ✅ Design for escalation (visual hierarchy)
7. ✅ UI must never lie (explicit failures)

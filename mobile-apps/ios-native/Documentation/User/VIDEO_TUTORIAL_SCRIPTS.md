# Fleet Management Video Tutorial Scripts

**Production Guide for Video Content Creators**

---

## Overview

This document contains complete scripts for five video tutorials covering essential features of the Fleet Management iOS app. Each tutorial includes:

- Full narration script with timing
- Screen recording notes and actions
- Visual callouts and annotations
- Recommended transitions and effects
- Total duration with timestamp breakdowns

### Video Specifications

- **Resolution**: 1080p (1920x1080) minimum
- **Frame Rate**: 30 fps
- **Audio**: Clear narration with background music at -20dB
- **Format**: MP4 (H.264)
- **Aspect Ratio**: 16:9 for web, 9:16 for mobile-first
- **Captions**: Include closed captions for accessibility

### Branding Guidelines

- Use app color scheme (blue primary, white/gray backgrounds)
- Include app logo in corner
- Use consistent fonts (SF Pro for iOS feel)
- End each video with call-to-action

## Production Notes

### General Video Guidelines

**Recording Environment**:
- Use screen recording software with high-quality capture (QuickTime, ScreenFlow, or similar)
- Record at native resolution (1242x2688 for iPhone Pro Max, scale to 1080p in post)
- Clean, decluttered iPhone home screen for professional look
- Use demo account with sample data pre-populated
- Disable notifications during recording (Do Not Disturb mode)

**Audio Recording**:
- Use high-quality microphone (preferably condenser mic)
- Record in quiet environment with minimal echo
- Speak clearly and at moderate pace
- Pause between sections for easier editing
- Record room tone for audio editing

**Editing**:
- Add background music at low volume (-20dB to -25dB)
- Use upbeat, professional royalty-free music
- Apply smooth transitions between scenes (cross dissolve, 0.5-1 second)
- Add text overlays for key points
- Use callout animations (circles, arrows, highlights)
- Include captions for accessibility
- Add zoom effects for small UI elements
- Speed up repetitive actions (form filling, scanning) to 2x-3x
- Export at 1080p, 30fps, H.264 codec

**Branding**:
- App logo watermark in corner (20% opacity)
- Consistent color scheme (blue primary, white/gray accents)
- Use app's color palette for highlights and callouts
- Include intro bumper (3 seconds) and outro (5 seconds)
- Social media handles and website in outro
- Subscribe button animation at end

**Accessibility**:
- Include closed captions with proper timing
- Use high-contrast text overlays
- Describe visual actions verbally
- Provide transcripts in video description
- Ensure audio is clear and at consistent volume
